# Automated-Weekly-Sales-Report-System-Google-App-Script-
This project automates the generation and email delivery of weekly sales performance reports using Google Apps Script
Problem Statement
Manual report generation is repetitive, time-consuming, and prone to errors.
Solution
Built a time-driven automation script that:
Pulls data from Sales Dashboard
Generates Google Docs report
Converts to PDF
Emails report automatically
Runs every Monday at 9AM
Technologies Used
Google Sheets
Google Apps Script
Google Docs
Gmail automation
Time-driven triggers
Automation Logic
Function: sendWeeklySalesReport
Trigger: Weekly (Monday 9AM)
Output: PDF report
Delivery: Automated email
